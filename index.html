Welcome to the Algorithmia Developer Center.

The Algorithmia platform allows you to deploy your machine and deep learning models at scale and provides a centralized repository for you and your team's models. Also check Algorithmia's open marketplace where you can call over 5,000 algorithms and microservices with our REST API. Below you'll find everything you need to get started.

Language Clients
  import Algorithmia
  input = "YOUR_USERNAME"
  client = Algorithmia.client("YOUR_API_KEY")
  algo = client.algo("demo/Hello/")
  print(algo.pipe(input))
